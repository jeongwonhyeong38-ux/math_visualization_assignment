# Math Visualization Assignment

This repository contains Python scripts and generated plots for visualizing mathematical functions, custom equations, student test data, and predictive modeling using regression.

## Project Description
The project demonstrates data processing and data visualization techniques using Python. It covers plotting basic functions, combined continuous math curves, discrete data representation (scatter, histogram, bar charts), and implementing a simple linear regression using `numpy.polyfit`.

## Libraries Used
* **NumPy**: Used for generating numeric arrays, computing linear spacing, and executing polynomial fitting.
* **Matplotlib (pyplot)**: Used for generating, customizing, and exporting various types of data visualizations.
